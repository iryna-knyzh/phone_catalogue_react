# Phone Catalog

A responsive e-commerce catalog for mobile phones, tablets, and accessories. Users can browse products by category, view detailed product pages, search and paginate through listings, and manage a shopping cart and favorites list.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/iryna-knyzh/phone_catalogue_react.git
cd phone_catalogue_react
```

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm start
```

## Features

- Home page with a banner slider and product categories
- Product listings for phones, tablets, and accessories
- Product details page with image gallery and specifications
- Search, sorting, and pagination
- Shopping cart with quantity management
- Favorites list
- Responsive layout for mobile, tablet, and desktop
